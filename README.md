# Networking

## IPv4 analyzer

### How to run: 
`python IPanalyzer.py`

Script will return you information about:

* Class of the IP addres (A-E)            
* Network address    
* Network broadcast  
* Subnet mask        
* Subnet address     
* Subnet broadcast   
* Host adresses range         
* Possible hosts (number)  

### Warning
There's no validating for input (yet) so be sure to porvide correct data
